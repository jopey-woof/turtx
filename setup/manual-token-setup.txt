Step 1: Create Long-Lived Access Token
=====================================

1. On your computer (not the touchscreen), open a web browser
2. Go to: http://10.0.20.69:8123  
3. Login with:
   - Username: shrimp
   - Password: shrimp

4. Once logged in:
   - Click your profile icon (bottom left)
   - Scroll down to "Long-lived access tokens"
   - Click "CREATE TOKEN"
   - Name: "Turtle Kiosk Token"
   - Click "OK"
   - COPY the long token that appears (very important!)

5. Run this command on the turtle machine:
   echo "KIOSK_TOKEN=YOUR_COPIED_TOKEN_HERE" >> /home/shrimp/turtle-monitor/.env

6. Then restart the kiosk:
   sudo systemctl restart kiosk.service

The touchscreen will then auto-login!
